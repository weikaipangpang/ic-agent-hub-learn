
# IC Agent Hub Learn

A learning and resource repository for **IC Agent Hub** tools and methodologies. This is a temporary storage repository for studying the IC agent tools recently released by Zhongke Qixin (中科麒芯), containing project examples and documentation for MCU, frontend, and backend development.

## Overview

IC Agent Hub provides tools and frameworks for developing embedded systems, EDA (Electronic Design Automation) environments, and associated workflows. This repository serves as a centralized learning hub with:

- **Frontend development** resources (complete)
- **MCU development** setup and guides (in progress)
- **Backend development** resources (in progress)
- **EDA environment** configurations
- **RTL design** assets for frontend applications

> **Note:** The official IC Agent Hub website (https://www.icagenthub.com) currently features three main projects. For complete and up-to-date resources, it is recommended to download directly from the official website.

## Getting Started

### For Frontend Development
1. Navigate to `IC-AGENT-HUB/frontend-eda-env-setup/`
2. Follow the environment setup instructions
3. Review `IC-AGENT-HUB/frontend-workflow-guide/` for workflow best practices
4. Reference RTL design files in `IC-AGENT-HUB/RTL-frontend-zip/`

### For MCU Development
1. Extract or navigate to `IC-AGENT-HUB/mcu-eda-env-setup/`
2. Follow the environment configuration guide
3. Review `IC-AGENT-HUB/mcu-workflow-guide/` for development workflows
